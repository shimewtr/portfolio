GitHub ActionsとGitHub Pagesを用いたOGP自動更新ツール

主な使用技術

AtCoder OGP

GitHub ActionsとGitHub Pagesを用いたOGP自動更新ツール

作成理由

OGPとTwitterの固定ツイートを利用して、表現できる情報を拡張できないかと考えたことと、
AtCoderユーザーの多くがレーティング画像を毎回撮影して、ツイートしていることがわかり、
レーティングの画像を自動で撮影し手間を省くのと同時に、
固定ツイートとOGPを活用した新しい情報領域を生成できると考え作成しました。

特徴

AtCoder OGPはGitHub Pages、GitHub Actions、Dockerなどを利用して作成されています。
互換性やユーザーの開発環境などを考慮せず、同一の処理が行えます。

利用者は関連技術の知識がなくても、自身のアカウント名をコードに記載するだけで、
自身のレーティング画像がOGPに設定されているGitHub Pagesを作成できます。
GitHub PagesのURLをツイートするだけで自動で更新されるレーティングを共有することができます。
